# 模板集：场景图提示词 · 15秒精简I2V · 本集存档 · 连载状态 · 角色档案

## 一、场景图提示词模板（发给生图工具，画面无主要角色）·精简电影化风格（实拍验证质感最佳）
```
生成图片设计电影化场景，不要出现主要角色，16:9，realistic，photorealistic，4k，
razor sharp focus，high clarity，natural【时段】lighting，clear shadows，crisp details。
场景是【一句话场景】，写实摄影质感：
【视觉层次一，短行】；
【视觉层次二，短行】；
【视觉层次三——关键功能区，动线用括号简注】。
色调以【主色组】为主，【点缀色】做点缀。
```
**结构纪律**：短行分层（每行一个层次）、只留必要元素、不写冗长功能清单、不堆形容词；背景NPC一行带过（需要人气的场景可含背景路人，主要角色永不入底图）；插画风项目把写实词串换成`clean illustration，smooth shading，4k，high clarity，refined edges`（两系勿混写）。
挑构图按"首帧即封面"标准（hook-retention.md）。
同一地点再次出场：把旧场景图一并发给生图工具，只改光线/时间/角度（"基于附图这间房间，改为清晨…"）。

## 二、15秒精简I2V模板（默认输出格式）
```
第一人称我视角，真实电影质感。场景按@图片1（首帧），【角色名】按@图片2（多角色依次@图片3…）。
【多角色时加站位表：站位：A在我左前方；B正前方偏右。】
0-3秒：【我的一个动作＋镜头方向】（0-2秒须命中钩子三选一，见hook-retention.md）。
3-6秒：【一个推进动作，物理细节】。
声音：【环境底＋关键音效】（放在有听觉反应的段，1-2处即可）
6-9秒：【冲突/爆发（转点），程度副词＋物理结果】。
【角色】（语言，语气标签）：「短句」
中文字幕行 / 英文字幕行（台词原语言=中文时省去中文字幕行）
9-12秒：【角色反应，体态化情绪】＋台词【我的台词同样带语气标签】。
12-15秒：【收尾构图（温柔停顿），镜头停在何处】。
背景音乐：【配器】，转点处音乐骤停只留【功能音效】，结尾渐弱只留【环境音】。（可选）
约束：全程第一人称，不出现我的全身和脸；【角色】外观与@图片2完全一致（含【标志性特征】，无特征时降级见SKILL.md冷启动）；
【角色】的动作一次一个（【动作链】）；我的手部动作一次一个（【动作链】）；
【场景】布局与@图片1一致；台词段镜头保持稳定；【场景转换次数声明，天气/光线中途变化用"无场景转换（××为同场景内变化）"】。
真实电影质感，realistic，photorealistic，neutral white balance，true-to-life colors，clean highlights，sharp focus，no noise，no grain，no artifacts，no yellow filter，no vintage look。
```
尾帧登记一句（下集衔接用）。

## 六、定稿前 Master Checklist（合并全库自检，逐条过）
**空间与动作**
1. 复述参考图静态信息了吗？（删）
2. 空话/明喻（像/仿佛）？（搜删，换可观察事实）
3. 每个动作有主语、触发源、终点？相邻段姿势能衔接？
4. 方位词全部来自看图记录？镜头转向都交代了？
5. 同段落指令冲突（镜头稳定+大幅运动）？
6. 手部：全集≥2次有功能动作、无凑数微操（他人为活动角色的段我的手可静态在场）？
7. 角色躯干保持拟人？大兽性动作≤1个且被拟人动作夹住？
