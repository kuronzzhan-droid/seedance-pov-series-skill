# POV多角色调度 · 明喻禁用 · 解释镜头消除 · 场景图两种用法

## 多角色调度四则
1. **扇形站位法**：我是圆心，角色分布在视野扇形（左前/正前/右前×远近两圈），提示词开头写**站位表**一次锚定全片。15秒POV内3人为舒适上限。
2. **一段一个活动角色**：每段只有一人大动作，其余给待机姿势（抱臂靠墙/趴桌沿）——"低功耗在场"；轮换活动权=多角色戏的节拍器。
3. **视线即调度**：我转头看谁谁才上镜，转头必须有触发；默认"谁说话我看谁"，高级档"说话人在画外，我看听话人的反应"——反应镜头比说话镜头值钱。入场排队：多人不同时入画。
4. **台词与身份**：总量仍≤4句、每人最多1句、允许有人全程不说话；每人一个身份锚+参考图绑定；禁集体代词，动作点名到人。

## 明喻清零（"像什么一样"双重毒药）
①文学腔AI味 ②污染生成——"像狗一样趴着"会真的画出狗。**删掉"像"，写动作本体**（姿势+速度+幅度）：
- ❌像在汇报任务一样郑重 → ✅挺直背，下巴微收，一字一顿
- ❌像接圣物一样 → ✅双手平举接过，指尖并拢，放低了呼吸
台词里角色说比喻允许（角色语言）；正文自检：搜"像/仿佛"逐个清除。

## 解释镜头消除法
解释镜头=为"让观众懂"而存在、角色没有动机的镜头。对策：
1. 信息挂在动作上（交代环境靠"我做事路过"，不用环视）
2. 道具用因果引入（`他的耳朵猛地转向桌面，我顺着看过去：契约在发光`）
3. 设定靠后果呈现（不解释怕吹风机，直接拍炸毛）
4. 首帧即交代（场景图给过的信息，正文永不复述）
5. 检查法：每个镜头问"**我此刻为什么看这里**"——答案必须是角色动机，不能是"观众需要知道"。

## 场景参考图两种用法
**模式A·首帧模式**：图=我的POV第一帧，构图锁死零漂移——静态开场/对话戏首选。绑定：`开场画面构图与@图片1完全一致（首帧）`
**模式B·场景参考模式（入座法）**：图=布局字典（非我的视角），适合移动动线/多角色分布广的集：
1. 入座声明：定"我在图中哪个位置、面向哪"
2. 坐标翻译表：图的绝对方位→我的相对方位（图里沙发在左、我面向沙发→沙发=我正前方），逐地标翻译，正文只用翻译后的方位
3. 绑定语法：`@图片1为场景布局与陈设参考（非首帧）；镜头为第一人称，我位于【图中位置】、面向【方向】`
4. 开场锚定句三件套：我在哪+面向哪+看到什么
风险对策：B漂移风险高——可加**文字首帧**补锁（`开场第一帧：从门口望向室内，左侧近处是沙发扶手`）。
多角色×B：站位表用图内地标定位再翻译（`他站在图中壁炉前=我的左前方近处`）——地标锚定比纯方位稳。
